##  Netflix Data Analytics Dashboard  ##

 ## Overview
Netflix Data Analytics Dashboard is an interactive web application built using Python, Pandas, Matplotlib, and Streamlit.
The project analyzes Netflix movies and TV shows data from a CSV dataset and presents meaningful insights through visualizations and statistics.
The dashboard helps users explore Netflix content by analyzing content types, ratings, release trends, genres, and country-wise distribution.

 ## Features
- Movies vs TV Shows Comparison
- Content Ratings Analysis
- Release Year Trend Analysis
- Genre Distribution Analysis
- Country-wise Content Distribution
- Interactive Data Visualizations
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
  
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit

 ## Dataset
The project uses the Netflix Titles Dataset, which contains information about Netflix content including:- 
- Title
- Type (Movie / TV Show)
- Genre
- Rating
- Release Year
- Country
- Duration
- Description
- Key Insights

## The dashboard provides insights such as:
- Movies vs TV Shows
Compare the overall distribution of Movies and TV Shows available on Netflix.

- Content Ratings
Analyze the most common ratings such as TV-MA, TV-14, PG-13, and others.

- Content Growth Over Years
Visualize how Netflix content has expanded over different years.

- Genre Analysis
Identify the most popular genres available on Netflix.

- Country-wise Distribution
Explore which countries contribute the most content to the Netflix platform.

Installation
Clone the repository:
git clone https://github.com/your-username/Netflix-Data-Analytics-Dashboard.git

Move to the project directory:
cd Netflix-Data-Analytics-Dashboard

Install required packages:
pip install -r requirements.txt

Run the application:
streamlit run main.py
